# Justâ˜…Recipes
Recipe site. The twist is that it only has recipes, like it should!
No cookies. No ads. No life stories. No user accounts. No javascript. Just recipes! 
Currently, recipes are only added via pull request. **Might** add other options in the future.
Recipes are stored as plain text files in the src directory.

## Submission
Helping is simple, see [submitting](https://github.com/MaksurTM/recipes/wiki/Submitting).
You can also improve other ones and correct errors if you feel the need to.

## Goals
- Thousands of recipes.
- A good way to sort recipes.
- A better mimimalist aesthetic.

## Features
- The website's full source code is always available to the public here on github. 
- Recipes can be sent via pull requests.
- Static website generation, all recipes are sent as markdown files and, later, converted into html files.
- 0% Javascript, might add a bit for search bar in the future.
